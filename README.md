# 효율성을 중시하는 백엔드 개발자 황가연입니다.

## 유연한 개발자입니다.
1. 문제 상황을 조기에 발견하고 해결한 경험이 있습니다.
2. 새로운 분야에 망설임 없이 도전하고 성취한 경험이 있습니다.
3. 기존의 방식에 집착하지 않고, 상황에 맞는 더 나은 방법을 빠르게 수용합니다.
4. 기술보다 **문제의 본질**을 먼저 보는 개발을 지향합니다.

## 성장할 수 있는 팀을 선호합니다.
1. 코드 리뷰와 피드백이 자연스럽게 오가는 팀
2. 도전적인 목표를 가진 팀
3. 책임과 자율이 균형 잡힌 환경
